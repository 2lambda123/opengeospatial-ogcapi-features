[[req_optimistic-locking_last-modified]]
[width="90%",cols="2,6a"]
|===
^|*Recommendation {counter:rec-id}* |*/rec/optimistic-locking/last-modified*

The response to a HTTP GET operation used to retrieve a representation of a resource SHOULD include a https://datatracker.ietf.org/doc/html/rfc7232#section-2.2[`Last-Modified`] header representing the last time the origin server believes that the selected representation of the resource was modified.
|===
