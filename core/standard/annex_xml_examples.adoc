== XML examples (Informative)

=== Overview

This annex includes examples of XML/GML responses to illustrate how the
OpenAPI fragments used to define the requirements for the
<<rc_core,Core requirements class>> are expressed in XML using the
link:https://raw.githubusercontent.com/opengeospatial/WFS_FES/master/core/xml/wfs.xsd[WFS 3.0 Core XML Schema].

The service used in the examples has a single feature collection (airports).
